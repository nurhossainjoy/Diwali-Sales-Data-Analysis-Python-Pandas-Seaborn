This project presents an exploratory data analysis (EDA) of Diwali sales data using Python. 
The objective is to identify key customer segments, purchasing patterns, and product trends 
that drive revenue during the Diwali festive season.

The analysis covers customer demographics, occupation-wise spending, state-wise performance, 
gender comparisons, and product category insights, supported with visualizations.
